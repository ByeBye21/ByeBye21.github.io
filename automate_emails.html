📧 Email Automation Pro

Professional Email Campaign Management Tool

A comprehensive PyQt5-based desktop application for automating email campaigns with modern GUI, CSV management, and advanced template personalization.

🚀 Key Features

  • Modern GUI Interface - Professional card-based design with sidebar navigation
  • SMTP Authentication - Support for Gmail, Outlook, Yahoo, iCloud, and custom servers
  • Flexible CSV Management - Any column structure supported with smart email detection
  • Template Personalization - Dynamic {{attribute}} insertion with dropdown selectors
  • Bulk Email Campaigns - Progress tracking and real-time statistics
  • File Attachments - Multiple file support with size display
  • Activity Logging - Comprehensive logs with HTML export
  • Configuration Management - Save/load email settings and templates

💻 Application Architecture


EmailAutomation/
├── email_gui.py              # Main GUI Application
│   ├── EmailAutomationApp    # Main window class
│   ├── SetupPage            # Account configuration
│   ├── RecipientsPage       # CSV management
│   ├── ComposePage          # Email composition
│   ├── SendPage             # Campaign dashboard
│   └── LogPage              # Activity monitoring
├── email_app.py              # Backend Engine
│   ├── EmailApplication     # Main app logic
│   ├── EmailSender          # SMTP handling
│   ├── CSVReader           # CSV processing
│   └── ConfigurationManager # Settings management
├── config.json              # Configuration storage
├── recipients.csv           # Default CSV file
└── requirements.txt         # Dependencies
