Email Extractor

A powerful Python tool for extracting emails, contact information, and social profiles from websites using advanced crawling techniques, JavaScript rendering, and AI-powered validation.

Features

  • Multi-method email detection (standard patterns, mailto links, obfuscated text, JavaScript-generated content)
  • Full JavaScript support for SPAs and React websites using Playwright
  • Contact intelligence for names, titles, companies, phone numbers, and social profiles
  • Real-time email verification and data quality scoring
  • OCR extraction for emails in images and documents
  • Multi-language support for international websites (Turkish, Spanish, etc.)
  • Multiple output formats (CSV, JSON, Excel) with detailed analytics
  • Batch processing for handling multiple domains simultaneously

Application Modules

  • Main CLI: Command-line interface for initiating crawls
  • Website Crawler: Async web crawling engine with Playwright support
  • Email Extractor: Multi-method email extraction engine
  • Contact Matcher: AI-powered contact information association
  • Utilities: Configuration, logging, patterns, text processing, validation, and export modules

Technologies Used

  • Python with Playwright for web crawling
  • re module for regex-based pattern matching
  • pytesseract for OCR extraction
  • pandas for data handling and Excel output
  • pydantic for configuration management
  • Asyncio for efficient batch processing
  • JSON and CSV modules for flexible output formats

Installation & Usage


# Install dependencies
python install.py

# OCR Setup (Optional)
# Windows
choco install tesseract
# or
conda install -c conda-forge tesseract

# Ubuntu/Debian
sudo apt install tesseract-ocr

# macOS
brew install tesseract

# Verify Tesseract
tesseract --version

# Basic Usage
python main.py --url https://example.com

# Batch processing
python main.py --urls-file websites.txt --output excel

# JavaScript-heavy sites
python main.py --url https://example.com --use-javascript

# High-quality extraction
python main.py --url https://example.com --validate-emails --extract-social --output excel
